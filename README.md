# Arkanoid SFML
Prosta gra Arkanoid w C++ z wykorzystaniem SFML.

## Scenariusz
Gracz steruje paletką, odbija piłkę i niszczy kolorowe bloki. Gra kończy się, gdy piłka wypadnie poza ekran. Wyniki są zapisywane i można je obejrzeć w menu.

## Funkcjonalności
- Menu: Nowa gra, Ostatnie wyniki, Wyjście  
- Sterowanie paletką: A/D lub strzałki  
- Bloki o różnym kolorze i HP  
- Odbicia piłki od paletki, bloków i ścian  
- Zapis i przywracanie stanu gry (GameState)  
- Wyświetlanie ostatnich 5 wyników  

## Uruchomienie
1. Skonfiguruj SFML i ścieżkę do czcionki Arial.  
2. Skompiluj projekt.  
3. Uruchom program i wybierz opcję w menu.  

## Sterowanie
- A / Strzałka w lewo — ruch w lewo  
- D / Strzałka w prawo — ruch w prawo  
- Enter — wybór w menu  
- Esc — powrót do menu
